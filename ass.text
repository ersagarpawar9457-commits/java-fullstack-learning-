Example with source code 

1. Two Sum (LC 1)
Description:
 Find indices of two numbers in an array that add up to a target.
Example:
 Input: [2,7,11,15], target = 9
 Output: [0,1]
Approach (HashMap):
Store number → index in a HashMap
For each element, check if target - num exists
Time: O(n)

2. Group Anagrams (LC 49)
Description:
 Group strings that are anagrams of each other.
Example:
 Input: ["eat","tea","tan","ate","nat","bat"]
 Output: [["eat","tea","ate"], ["tan","nat"], ["bat"]]
Approach (HashMap + ArrayList):
Sort characters of each word to create a key
Map key → list of words
Return map values

3. Top K Frequent Elements (LC 347)
Description:
 Return the k most frequent elements.
Example:
 Input: [1,1,1,2,2,3], k = 2
 Output: [1,2]
Approach (HashMap + PriorityQueue):
Count frequency using HashMap
Use max heap based on frequency
Extract top k

4. Valid Parentheses (LC 20)
Description:
 Check if brackets are balanced.
Example:
 Input: "()[]{}"
 Output: true
Approach (Stack / Deque):
Push opening brackets
Pop and match on closing brackets
Stack must be empty at end

5. Longest Substring Without Repeating Characters (LC 3)
Description:
 Find length of longest substring with unique characters.
Example:
 Input: "abcabcbb"
 Output: 3
Approach (HashSet + Sliding Window):
Maintain window with HashSet
Expand right, shrink left on duplicates
Track max length

6. Merge Intervals (LC 56)
Description:
 Merge overlapping intervals.
Example:
 Input: [[1,3],[2,6],[8,10]]
 Output: [[1,6],[8,10]]
Approach (ArrayList):
Sort intervals by start
Merge overlapping ones into result list

7. Subarray Sum Equals K (LC 560)
Description:
 Count subarrays whose sum equals k.
Example:
 Input: [1,1,1], k = 2
 Output: 2
Approach (HashMap):
Use prefix sum
Map prefixSum → frequency
Check currentSum - k

8. Find All Duplicates in an Array (LC 442)
Description:
 Return numbers appearing twice.
Example:
 Input: [4,3,2,7,8,2,3,1]
 Output: [2,3]
Approach (HashSet / ArrayList):
Track seen numbers in HashSet
Add duplicates to result list

9. Kth Largest Element in an Array (LC 215)
Description:
 Find the kth largest element.
Example:
 Input: [3,2,1,5,6,4], k = 2
 Output: 5
Approach (PriorityQueue):
Maintain min heap of size k
Remove smallest when size exceeds k

10. LRU Cache (LC 146)
Description:
 Design a cache with O(1) get and put.
Example:
 Operations: put(1,1), put(2,2), get(1)
Approach (LinkedHashMap):
Use access-order LinkedHashMap
Override removeEldestEntry

11. Intersection of Two Arrays (LC 349)
Description:
 Find unique common elements.
Example:
 Input: [1,2,2,1], [2,2]
 Output: [2]
Approach (HashSet):
Add first array to set
Check second array for presence

12. Sort Characters by Frequency (LC 451)
Description:
 Sort characters based on frequency.
Example:
 Input: "tree"
 Output: "eert"
Approach (HashMap + PriorityQueue):
Count character frequency
Use max heap based on frequency

13. Design HashSet (LC 705)
Description:
 Implement a basic HashSet.
Example:
 add(1), contains(1) → true
Approach (ArrayList / LinkedList):
Use array of buckets
Handle collisions using chaining

14. Minimum Window Substring (LC 76)
Description:
 Find the smallest substring containing all characters of t.
Example:
 Input: "ADOBECODEBANC", "ABC"
 Output: "BANC"
Approach (HashMap + Sliding Window):
Count needed characters
Expand and contract window
Track minimum length

15. Evaluate Reverse Polish Notation (LC 150)
Description:
 Evaluate postfix expression.
Example:
 Input: ["2","1","+","3","*"]
 Output: 9
Approach (Stack):
Push numbers
Pop operands on operator
Push result

16. Word Pattern (LC 290)
Description:
 Check if string follows a pattern.
Example:
 Input: "abba", "dog cat cat dog"
 Output: true
Approach (HashMap + HashSet):
Map pattern char → word
Ensure one-to-one mapping

17. Continuous Subarray Sum (LC 523)
Description:
 Check if subarray sum is multiple of k.
Example:
 Input: [23,2,4,6,7], k = 6
 Output: true
Approach (HashMap):
Store remainder → index
Check repeating remainder with distance ≥ 2

18. Queue Reconstruction by Height (LC 406)
Description:
 Reconstruct the queue based on height and position.
Example:
 Input: [[7,0],[4,4],[7,1]]
Approach (ArrayList):
Sort by height descending
Insert at index k

19. Frequency Stack (LC 895)
Description:
 Stack that pops most frequent element.
Example:
 Push: 5,7,5,7,4,5 → Pop: 5
Approach (HashMap + Stack):
Map value → frequency
Map frequency → stack
Track max frequency

20. Daily Temperatures (LC 739)
Description:
 Find days until a warmer temperature.

 Input: [73,74,75,71,69,72,76,73]
