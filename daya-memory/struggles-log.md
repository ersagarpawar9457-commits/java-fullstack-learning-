## Common Java Gotchas (Reference)

| Issue | What Happens | Solution |
|-------|--------------|----------|
| NullPointerException | Accessing method on null object | Always check for null, use Optional |
| ConcurrentModificationException | Modifying collection while iterating | Use Iterator.remove() or streams |
| StringBuffer vs StringBuilder | Confusion on when to use | StringBuffer = thread-safe (slower), StringBuilder = not thread-safe (faster) |
| == vs .equals() | Comparing objects incorrectly | == compares reference, .equals() compares content |
| Spring @Autowired NPE | Bean not injected | Don't use `new`, check annotations |
